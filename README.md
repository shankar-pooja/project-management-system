# Project Management System

## Features
- User Registration
- User Login
- Dashboard
- Project Management
- Task Management
- MySQL Database Integration
- REST APIs using Express.js

## Tech Stack
Frontend:
- React.js
- React Router
- Axios

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Installation

Frontend:
